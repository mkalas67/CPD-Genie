# **App Name**: ASO Assist

## Core Features:

- Document Upload: File upload via browser to ingest training documents (PDF, DOCX, PPTX, TXT).
- Context Input: Input field for specifying the target country and industry for the ASOs.
- ASO Generation: AI-powered suggestion tool for generating Aims, Skills, and Outcomes (ASOs) based on the uploaded content and specified context, tailored to be context aware.
- Clarification Prompts: The tool will present clarifying questions based on identified gaps or ambiguities in the uploaded materials and context inputs, if necessary.
- ASO Display: Display the generated ASOs in a clear and structured format, divided into Aims, Skills, and Outcomes.
- Copy to Clipboard: Copy buttons for each generated ASO to easily copy the text to the clipboard.
- Basic Analytics: Basic analytics logging (user, timestamp, country/industry, number of documents uploaded).

## Style Guidelines:

- Primary color: A saturated blue (#29ABE2) to evoke feelings of trust and competence, reflecting the AI's helpfulness.
- Background color: Light blue (#E5F6FD) for a clean and professional look, suitable for a tool used by trainers.
- Accent color: A violet color (#8D29E2) will draw attention to key actions (like the 'Copy' button).
- Body: 'PT Sans' (sans-serif) for a friendly feel; Headlines: 'Playfair' (serif) to give a fashionable, high-end feel.
- Code font: 'Source Code Pro' for displaying any code snippets, instructions or analytics.
- Use clear, professional icons for actions like upload, copy, and help, ensuring ease of use.
- A clean and intuitive layout, with clear sections for input, questions, and ASO results.
